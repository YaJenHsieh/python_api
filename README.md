# python 串接食藥署 API
運用
* Flask
* PostgresSQL(MySQL語法存入資料庫)
* Bootstrap

功能
* 新增 : 新增文章的標題、內容、網址
* 刪除 : 刪除網站內已有的文章
* 修改 : 編輯每一篇的文章標題、內容
* 查詢 : 以標題、內容關鍵字收尋所有文章

Heroku 雲端主機部署網站
